# 📊 Mobile Sales Dashboard (Power BI)

## 📌 Project Overview
This project presents an interactive Power BI dashboard built to analyze mobile sales data. The dashboard helps in understanding key business metrics, identifying trends, and gaining insights into customer behavior and sales performance.

## 🎯 Objectives
- Analyze total sales, quantity, and transactions
- Understand customer preferences and ratings
- Identify top-performing cities, brands, and mobile models
- Track monthly and daily sales trends
- Explore payment method distribution

## 📊 Dashboard Features
- 📍 Sales by City (Geographical Analysis)
- 📈 Monthly Sales Trend
- ⭐ Customer Ratings Distribution
- 💳 Transactions by Payment Method
- 📱 Sales by Mobile Brand & Model
- 📅 Sales by Day Name
- 🔍 Interactive Filters (Brand, Model, Payment Method, Day)

## 📈 Key Metrics
- Total Sales: 769M+
- Total Quantity Sold: 19K+
- Total Transactions: 4K+
- Total Customers: 40K+

## 🛠 Tools & Technologies Used
- Power BI
- DAX (Data Analysis Expressions)
- Data Cleaning & Transformation
- Data Modeling
